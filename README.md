# pyqt_making_dynamic_gui_with_gpt_json_response_example
Making dynamic GUI based on GPT response

## Requirements
* PyQt5 >= 5.14
* openai

## Detailed Description
Using GPT to make GUI dynamically, enough said :)

## How to Install
1. git clone ~
2. pip install -r requirements.txt
3. python main.py
